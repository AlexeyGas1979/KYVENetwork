## About

The KYVE + Cosmos node allows you to bridge any data stream from a Cosmos chain onto Arweave.

## Usage

```js
import CosmosInstance from "@kyve/cosmos";

CosmosInstance("Cosmos // Mainnet", jwk).run();
```
