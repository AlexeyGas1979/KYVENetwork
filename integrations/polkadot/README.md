## About

The KYVE + Polkadot node allows you to bridge any data stream from a Polkadot substrate onto Arweave.

## Usage

```js
import PolkadotInstance from "@kyve/polkadot";

PolkadotInstance("Polkadot // Mainnet", jwk).run();
```
